## Set up

### Client

```
cd client
npm i
```

### Server

```
cd server
npm i
```

## Run

Your can run both the client and server concurrently using

```
npm run dev
```

### Client

```
cd client
npm run dev
```

### Server

```
cd server
npm start
```

## Extra

Run codegen

```
cd client
npm run codegen
```
